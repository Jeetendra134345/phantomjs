Phantomjs +Plugins

This is a fork of the phantomjs project with plugin support added back in. That's it. Enjoy!

Additional Dependencies

In addition to the main branch dependencies, you're going to need: *libglib2.0-dev *libx11-dev *libxext-dev *libfreetype6-dev *libxcursor-dev *libxrandr-dev *libxv-dev *libxi-dev *libgstreamermm-0.10-dev *xvfb (at runtime)
